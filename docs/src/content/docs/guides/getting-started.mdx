# Getting Started

## Installation

1. Download the appropriate installer for your operating system from the [releases page](https://github.com/0-don/clippy/releases)
2. Run the installer
3. Launch Clippy from your applications menu or system tray

## First Launch

- The application will start minimized in your system tray
- Click the tray icon to open the main window
- Default hotkey to show/hide: `Ctrl+Y` or `Ctrl+D`

## Basic Usage

### Clipboard Management

1. **Copying Content**

   - Copy any content (text, images, files) as usual
   - Clippy automatically saves it to your history

2. **Accessing History**

   - Click the tray icon or use the hotkey
   - Your clipboard history appears in the main window
   - Click any item to copy it back to your clipboard

3. **Searching History**

   - Use the search bar to find specific content
   - Smart search recognizes:
     - Links
     - Colors
     - Images
     - Hex codes

4. **Managing History**
   - Star items to add them to favorites
   - Clear history by type (text, images, files)
   - Delete individual items

### Advanced Features

#### Type Out Mode

- Use `Ctrl+B` to type out content where pasting isn't allowed
- Useful for password fields or restricted input areas

#### Cloud Sync

1. **Setup**

   - Go to Settings → Sync
   - Click "Connect Google Drive"
   - Follow the authentication process

2. **Configuration**
   - Set sync limits for different content types
   - Choose what to sync (favorites, history)
   - Configure sync frequency

#### Security Features

1. **Encryption**

   - Enable encryption in Settings → Security
   - Set a strong password
   - Choose what content to encrypt

2. **Privacy Settings**
   - Configure content size limits
   - Set up replace patterns for sensitive data
   - Choose database location

### Customization

#### Interface

1. **Theme**

   - Toggle between light/dark mode
   - Adjust display scale
   - Choose your preferred language

2. **Window**
   - Drag to reposition
   - Resize as needed
   - Set default position

#### Keyboard Shortcuts

1. **Default Hotkeys**

   - Show/Hide: `Ctrl+Y` or `Ctrl+D`
   - Type Out: `Ctrl+B`

2. **Custom Hotkeys**
   - Go to Settings → Hotkeys
   - Click any function to set custom shortcut
   - Use "Clear" to reset to default

### System Integration

#### Autostart

- Enable in Settings → General
- Clippy will start automatically with your system

#### System Tray

- Right-click for quick actions
- Left-click to show main window
- Middle-click for additional options

## Troubleshooting

### Common Issues

1. **Hotkeys Not Working**

   - Check for conflicts with other applications
   - Verify permissions in system settings
   - Try changing hotkey combinations

2. **Sync Problems**

   - Check internet connection
   - Verify Google Drive permissions
   - Try reconnecting sync

3. **Performance Issues**
   - Clear old history
   - Adjust content size limits
   - Check available disk space

### Getting Help

- Check the [GitHub Issues](https://github.com/0-don/clippy/issues)
- Review the [Technical Documentation](/docs/technical)
- Submit a bug report if needed

## Tips and Tricks

1. Use favorites for frequently used content
2. Enable encryption for sensitive data
3. Customize hotkeys for your workflow
4. Use type-out mode for restricted fields
5. Set up cloud sync for backup
6. Configure size limits to manage storage
7. Use search to quickly find content
8. Clear history regularly for better performance
