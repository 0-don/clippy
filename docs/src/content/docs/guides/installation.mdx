## Prerequisites

Before you begin, ensure you have the following installed:

- [Rust](https://www.rust-lang.org/tools/install) (latest stable version)
- [Node.js](https://nodejs.org/en/download/) (v16+ recommended)
- [Tauri prerequisites](https://tauri.app/v1/guides/getting-started/prerequisites/)

> **Note:** Make sure the `cargo` and `rustup` commands are available in your shell. You can verify with `cargo --version`.

---

## Download / Clone

You can download the project from [GitHub](https://github.com/0-don/clippy) or clone it via:

```bash
git clone https://github.com/0-don/clippy.git
cd clippy
```

---

## Install Dependencies

After cloning the repository, navigate to the project directory and install the dependencies:

```bash
cd clippy
npm install
```

---

## Run the Application

Once the dependencies are installed, you can start the application with:

```bash
npm run d
```

---

## Tested Platforms

Clippy has been tested on the following platforms:

<ul>
  <li>Linux(x11) KDE Plasma (Disable Focus Stealing Prevention)</li>
  <li>Windows</li>
  <li>Mac (hotkeys needs to be changed in settings)</li>
</ul>
